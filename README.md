# 🚀 GESTOR DE PROYECTOS BIM - RAYMUNDO EDUARDO

Sistema personalizado de gestión de tareas y cronogramas para proyectos de infraestructura y minería. Desarrollado para optimizar el flujo de trabajo bajo estándares **ISO 19650**.

## 📊 VISTAS DEL SISTEMA
* **Tablero Kanban:** Control de flujo (Pendiente, En Progreso, Terminado).
* **Diagrama Gantt:** Visualización de cronogramas y frentes de trabajo.
* **Vista Lista:** Auditoría rápida de fechas y prioridades.

## 🛠️ TECNOLOGÍAS UTILIZADAS
* **Frontend:** HTML5, Tailwind CSS y JavaScript.
* **Backend:** Google Apps Script (GAS).
* **Base de Datos:** Google Sheets.
* **Hosting:** GitHub Pages.
